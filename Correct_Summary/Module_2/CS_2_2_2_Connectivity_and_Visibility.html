Information System Architecture
📖 ASCM Definition — Information System Architecture: a model of how the organization operates regarding information. Considers four factors: (1) information flow, (2) data and hardware, (3) applications, and (4) operations management and IT staff.
  • Database: structured repository of data. Managed by a DBMS (Database Management System).
  • Data dictionary: catalog of requirements/specifications for an IS, or a file that stores facts about data itself (metadata).
  • Network: interconnection of computers, terminals, and channels to facilitate file and peripheral sharing. LAN = local; WAN = wide area; VPN = secure internet tunnel.
  • Intranet: private network using internet technology, for internal use. Extranet: network connection to a partner’s network using secure internet protocols.

SaaS & Cloud Computing
📖 ASCM Definition — Software as a Service (SaaS): a software licensing and distribution model that provides access to applications via the internet on a subscription basis.
  • No local installation or download. Logic and data stored centrally; users access via browser.
  • Benefits: eliminates installation/upgrade time, lower upfront cost, always on latest version.
  • Cloud computing: IT resources (storage, compute, software) delivered over the internet. Mainstream since 2020. Supports SaaS, PaaS, and IaaS models.
  • SC software benefits from cloud: track-and-trace, supplier quality metrics, WMS inventory aggregation, API-based integration, automated regulatory forms.

E-Commerce & VMI
  • B2B e-commerce: electronic transactions between businesses. Includes EDI, electronic marketplaces, and web-based procurement portals.
  • B2C e-commerce: direct-to-consumer transactions via websites or apps.
  • VMI (Vendor Managed Inventory): the supplier monitors the customer’s inventory levels and replenishes automatically. Reduces stockouts and ordering workload for the customer.
  • Consignment: supplier owns inventory at the customer’s location until it is used/sold. Reduces customer working capital tied up in inventory.

TERM / CONCEPT
What is a LAN?
ANSWER
Local Area Network: a high-speed data communication system for linking computer terminals, printers, and databases in close proximity. Used within a building or campus.
TERM / CONCEPT
What is a WAN?
ANSWER
Wide Area Network: connects computers across geographically dispersed locations (multiple buildings, cities, countries). Used to share information between facilities.
TERM / CONCEPT
What is a VPN?
ANSWER
Virtual Private Network: a low-cost internet-based secure transmission method that allows secure data exchange over the public internet — acts like a private network.
TERM / CONCEPT
What is the difference between intranet and extranet?
ANSWER
Intranet: privately owned network using internet technology for internal use only (employees). Extranet: a network connection to a partner’s network using secure internet protocols — extends internal systems to select external parties.
TERM / CONCEPT
What is a DBMS?
ANSWER
Database Management System: software designed for organizing data and providing the mechanism for storing, maintaining, and retrieving data. Examples: Oracle, SQL Server, MySQL.
TERM / CONCEPT
What is a data dictionary?
ANSWER
Either (1) a catalog of requirements and specifications for an information system, or (2) a file that stores facts about data itself (metadata) — field names, data types, relationships.
TERM / CONCEPT
What are 3 advantages of SaaS?
ANSWER
1. Eliminates installation and upgrade time (always on latest version). 2. Lower upfront cost (subscription vs. license). 3. Accessible from anywhere via browser — no hardware to maintain.
TERM / CONCEPT
What SC benefits does cloud computing enable?
ANSWER
Real-time track-and-trace across partners, supplier quality metric tracking, automated regulatory forms, API-based integration, WMS inventory aggregation, and faster scalability of SC systems.
TERM / CONCEPT
What is VMI (Vendor Managed Inventory)?
ANSWER
A system where the supplier monitors the customer’s inventory levels and replenishes automatically when stock falls below a threshold. Reduces stockouts and customer ordering workload.
TERM / CONCEPT
What is consignment inventory?
ANSWER
The supplier owns the inventory while it is stored at the customer’s location. The customer pays only when they use or sell it. Reduces the customer’s working capital tied up in inventory.
TERM / CONCEPT
What is B2B e-commerce?
ANSWER
Electronic commerce between businesses: EDI transactions, electronic marketplaces, procurement portals, supplier catalogs. Automates purchasing, reduces transaction costs.
TERM / CONCEPT
What is B2C e-commerce?
ANSWER
Electronic commerce directly with consumers via websites or apps. Requires last-mile logistics, digital payment systems, and often individual-order fulfillment capabilities.
TERM / CONCEPT
What is client/server configuration?
ANSWER
The most common IT configuration — clients (PCs/devices) send requests to servers (which process and store data). The server provides a specific service to multiple client devices.
TERM / CONCEPT
What is the operating system (O/S)?
ANSWER
A set of software programs that control the execution of hardware and application programs. Manages resources (CPU, memory, storage). Examples: Windows, Linux, Unix, MacOS.
TERM / CONCEPT
What is application software?
ANSWER
Software controlled by an OS that fills specific computing needs (word processing, ERP, WMS, etc.). Runs on top of the operating system.
TERM / CONCEPT
What is the difference between SaaS and conventional software?
ANSWER
Conventional: user purchases license, installs on own hardware, manages upgrades. SaaS: vendor hosts and manages software; user pays subscription and accesses via internet — no local installation.
TERM / CONCEPT
How does connectivity between SC partners work technically?
ANSWER
Via networks (LAN/WAN/VPN), APIs, EDI, or web portals. An extranet or cloud platform allows multiple partners to exchange data securely. APIs are becoming the dominant integration method.
